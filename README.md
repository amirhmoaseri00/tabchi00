# نصب

```sh
git clone https://github.com/amirhmoaseri00/tabchi00
cd tabchi00
chmod +x bot
./bot install
# and send [y] to finish install
```
* * *
## ساخت ربات
```
./bot create
./bot 1
وارد کردن ایدی عددی سودو#
وارد کردن شماره ربات#
```
